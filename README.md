Overview

This project demonstrates the implementation of a Machine Learning Model for Spam Email Detection using Python and popular machine learning libraries. The objective of this task is to classify messages as Spam or Not Spam based on their text content.

The project uses Natural Language Processing (NLP) techniques to convert text data into numerical features and applies a Naive Bayes Classifier for prediction.

Features
Load and preprocess email/message dataset
Remove duplicate records
Convert categorical labels into readable classes
Transform text into numerical vectors using CountVectorizer
Split dataset into training and testing data
Train model using Multinomial Naive Bayes
Predict spam or non-spam messages
Evaluate model performance using accuracy score and classification report
Data visualization using Matplotlib
Technologies Used
Python
